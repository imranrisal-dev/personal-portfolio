# Python Fundamentals & OOP Assignment

This repository contains a collection of Python scripts demonstrating fundamental programming concepts, data structures, control flow, and Object-Oriented Programming (OOP) principles. 

The project includes solutions to specific analytical tasks alongside a practical demonstration of **Polymorphism**.

---

## 📌 Features & Task Descriptions

### 1. Assignment Tasks (`task1.py` to `task5.py`)
These scripts solve practical problem-solving tasks based on explicit guidelines:

*   **Task 1: Student Marks Analyzer (`task1.py`)**
    *   Removes duplicates from a given list of student marks.
    *   Calculates the average marks.
    *   Identifies the highest and lowest scores.
*   **Task 2: Even Number List (`task2.py`)**
    *   Generates a list of even numbers from a range using **list comprehension**.
    *   Outputs the filtered sequence efficiently.
*   **Task 3: Word Frequency Counter (`task3.py`)**
    *   Analyzes a text string to count the frequency of each individual word.
    *   Stores and structures the tracking dataset dynamically inside a dictionary.
*   **Task 4: Loop Control Program (`task4.py`)**
    *   Iterates and prints numbers from 1 to 15.
    *   Implements the `continue` statement to skip numbers divisible by 3.
    *   Utilizes the `break` statement to terminate execution instantly when hitting the number 13.
*   **Task 5: Student Result System (`task5.py`)**
    *   Parses a raw dataset of student names and corresponding marks.
    *   Filters and outputs names scoring above a specific threshold (> 80).
    *   Identifies the top-performing student (top scorer) mathematically.

### 2. OOP Concept Demonstration
*   **Polymorphism (`pholymorphism.py`)**
    *   Demonstrates core Object-Oriented Programming capability where different classes can define methods with identical names but custom behavior.
    *   Showcases how a clean unified interface can invoke contextual implementation across polymorphic object instances.
